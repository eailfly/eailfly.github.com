"使用Spacemacs管理hexo"

Posted on Tue 27 December 2016 in misc

安装

在private layer － package.el 中添加

    (hexo :location(recipe
                    :fetcher github
                    :repo "kuanyui/hexo.el"
                    ))

melpa 中的版本好像有问题，需要使用github中的版本

在private layer - func.el 中添加

    (defun hexo-my-blog ()
        (interactive)
        (hexo "~/my-blog/"))

Usage

M-x hexo 可以进入 hexo—mode
